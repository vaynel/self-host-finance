## 4. 발견된 이슈 및 수정 사항

### 4.1 수정 완료

| 이슈 | 원인 | 조치 |
|------|------|------|
| `GET /reports/monthly-summary` 500 | `func.case(..., else_=0)` — SQLAlchemy `func`에는 case 없음 | `sqlalchemy.case()` import 후 `case((cond, val), else_=0)` 사용 |
| Reports 카테고리 차트 undefined | API는 `{ category, amount }`, 프론트는 `{ name, value }` 기대 | `category`→`name`, `amount`→`value` 매핑 처리 |

### 4.2 점검 중 확인된 사항

| 항목 | 내용 |
|------|------|
| CORS | `CORS_ORIGIN`에 프론트엔드 Origin 포함 필요 (예: `http://localhost:8000`, `http://localhost:5173`) |
| bcrypt | 5.0.0과 passlib 비호환 → `bcrypt==4.3.0` 고정 |
| 업로드 | multipart/form-data (file, accountId) 필수, JSON만 보내면 400 |
