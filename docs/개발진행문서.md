# FinFlow 개발 진행 문서

> 최종 수정일: 2026-03-12

---

## 1. 개요

이 문서는 FinFlow 프로젝트의 개발 진행 상황을 추적합니다.

---

## 2. 전체 진행 현황

| 영역 | 진행률 | 상태 | 비고 |
|------|--------|------|------|
| 프론트엔드 | 100% | ✅ 완료 | React + Vite + Tailwind, mock 데이터 |
| 백엔드 API | 100% | ✅ 완료 | FastAPI 기반 REST API |
| 인증 | 100% | ✅ 완료 | JWT 로그인/회원가입/리프레시 |
| 거래·계좌·투자 | 100% | ✅ 완료 | CRUD, 리포트, 업로드 |
| DB 마이그레이션 | 100% | ✅ 완료 | Alembic 초기 스키마 |
| E2E 연동 | 0% | 🔲 대기 | 프론트 ↔ 백엔드 실제 연동 |

---

## 3. 백엔드 개발 현황

### 3.1 완료된 항목

| 구분 | 내용 |
|------|------|
| **프로젝트 구조** | FastAPI + SQLAlchemy + Pydantic |
| **인증 API** | `POST /v1/auth/login`, `register`, `refresh`, `logout` |
| **거래 API** | `GET/POST/PUT/DELETE /v1/transactions` (필터, 페이지네이션) |
| **계좌 API** | `GET/POST/PUT/DELETE /v1/accounts`, `GET /v1/accounts/:id/flow` |
| **투자 API** | `GET /v1/investments/holdings`, `GET/POST /v1/investments/trades`, `GET /v1/investments/prices/:ticker` |
| **리포트 API** | `GET /v1/reports/monthly-summary`, `GET /v1/reports/category-spending` |
| **업로드 API** | `POST /v1/upload/transactions` (CSV/Excel) |
| **설정 API** | `GET/PUT /v1/settings` |
| **DB 스키마** | users, refresh_tokens, transactions, accounts, investment_trades, user_settings |
| **마이그레이션** | `alembic/versions/001_initial.py` |
| **공통 응답** | `{ success, data, error, meta }` 형식 |
| **예외 처리** | 400/401/403/404/500 표준 에러 응답 |

### 3.2 미구현·추후 작업

| 항목 | 설명 |
|------|------|
| 주식 시세 API | `/investments/prices/:ticker` — 외부 API 연동 시 실제 시세 반환 (현재 빈 배열) |
| 파일 업로드 검증 | 파일 크기 제한(5MB 등), 확장자 화이트리스트 강화 |
| 비밀번호 재설정 | 이메일 인증 기반 비밀번호 초기화 (기획 문서 명시) |
| 테스트 | pytest 기반 단위·통합·API 테스트 작성 |

## 4. 실행 방법

### 4.1 사전 요구사항

- Python 3.11+
- PostgreSQL 15+
- 가상환경 권장

### 4.2 백엔드 실행

```bash
cd backend
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

`.env` 파일 설정:

```env
DATABASE_URL=postgresql://user:pass@localhost:5432/mandu
JWT_SECRET=your-secret-at-least-32-chars
CORS_ORIGIN=http://localhost:8000
```

DB 마이그레이션:

```bash
alembic upgrade head
```

서버 실행:

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8001
```

- API 문서: `http://localhost:8001/docs`
- 헬스 체크: `http://localhost:8001/health`

### 4.3 프론트엔드 실행

```bash
cd frontend
npm install
npm run dev
```

- 기본 URL: `http://localhost:8000`
