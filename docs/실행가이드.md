# FinFlow 실행 가이드

> 최종 수정일: 2026-03-12

---

## 포트 설정

- **Frontend**: `8000`
- **Backend**: `8001`

---

## 1. 사전 요구사항

### Backend
- Python 3.11 이상
- PostgreSQL 15 이상
- 가상환경 권장

### Frontend
- Node.js 18 이상
- npm 또는 yarn

---

## 2. Backend 실행 방법

### 2.1 환경 설정

```bash
cd backend
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2.2 환경 변수 설정

`backend/.env` 파일을 생성하고 다음 내용을 입력합니다:

```env
DATABASE_URL=postgresql://user:pass@localhost:5432/finflow
JWT_SECRET=your-secret-at-least-32-chars
CORS_ORIGIN=http://localhost:8000
PORT=8001
```

### 2.3 데이터베이스 마이그레이션

```bash
alembic upgrade head
```

### 2.4 서버 실행

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8001
```

또는 설정 파일의 포트를 사용하려면:

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port $(python -c "from app.config import get_settings; print(get_settings().port)")
```

### 2.5 확인

- API 문서: `http://localhost:8001/docs`
- ReDoc 문서: `http://localhost:8001/redoc`
- 헬스 체크: `http://localhost:8001/health`

## 3. Frontend 실행 방법

### 3.1 의존성 설치

```bash
cd frontend
npm install
```

### 3.2 개발 서버 실행

```bash
npm run dev
```

### 3.3 확인

- 기본 URL: `http://localhost:8000`

---

## 4. 전체 실행 순서

1. **PostgreSQL 데이터베이스 실행**
   ```bash
   # PostgreSQL이 설치되어 있다면
   sudo systemctl start postgresql  # Linux
   # 또는
   brew services start postgresql   # macOS
   ```

2. **Backend 실행**
   ```bash
   cd backend
   source .venv/bin/activate
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8001
   ```

3. **Frontend 실행** (새 터미널)
   ```bash
   cd frontend
   npm run dev
   ```

4. **브라우저에서 접속**
   - Frontend: `http://localhost:8000`
   - Backend API 문서: `http://localhost:8001/docs`

## 5. 문제 해결

### Backend 포트가 이미 사용 중인 경우

포트를 변경하려면:
- `backend/app/config.py`의 `port` 값을 변경하거나
- `.env` 파일에 `PORT=8001` 설정

### Frontend 포트가 이미 사용 중인 경우

포트를 변경하려면:
- `frontend/vite.config.ts`의 `server.port` 값을 변경

### CORS 오류가 발생하는 경우

- `backend/app/config.py`의 `cors_origin`이 frontend 포트(`http://localhost:8000`)와 일치하는지 확인
- `.env` 파일의 `CORS_ORIGIN` 설정 확인

## 6. 빌드 및 배포

### Frontend 빌드

```bash
cd frontend
npm run build
```

빌드된 파일은 `frontend/dist` 폴더에 생성됩니다.

### Backend 프로덕션 실행

```bash
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8001 --workers 4
```
